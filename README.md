# Анализатор расходов

Телеграм-бот для учёта личных расходов. Бот принимает траты, складывает их в базу
через небольшой API и показывает сводку по категориям.

## Архитектура

```
Telegram-бот (aiogram) → FastAPI → SQLite
```

## Запуск

1. Установить зависимости: `pip install -r requirements.txt`
2. Скопировать `.env.example` в `.env` и указать токен бота от @BotFather.
3. Запустить API: `uvicorn api.main:app`
4. В отдельном терминале запустить бота: `python -m bot.main`

## Как пользоваться

\- `/start` - краткая справка.
\- Сообщение вида `300 еда` - записывает расход (первое слово - сумма, остальное - категория).
\- `/stats` - сводка по категориям и итог.

## API

\- `POST /expenses` - `{user_id, amount, category}`, сохраняет расход.
\- `GET /stats?user_id=` - сумма по категориям и общий итог.
\- `GET /health` - проверка.
