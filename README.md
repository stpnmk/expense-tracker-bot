# Анализатор расходов

Телеграм-бот для учёта личных расходов. Бот принимает траты, складывает их в базу
через небольшой API, считает статистику по периодам и следит за месячными лимитами.

## Архитектура

```
Telegram-бот (aiogram) → FastAPI → SQLite
```

## Запуск

1. Установить зависимости: `pip install -r requirements.txt`
2. Скопировать `.env.example` в `.env` и указать токен бота от @BotFather.
3. Запустить API: `uvicorn api.main:app`
4. В отдельном терминале запустить бота: `python -m bot.main`

## Команды бота

\- `/start` - справка.
\- `300 еда` - быстрый ввод траты (первое слово - сумма, остальное - категория).
\- `/add` - добавить трату через кнопки выбора категории.
\- `/stats [today|week|month|all]` - сводка по категориям (по умолчанию за месяц).
\- `/last` - последние траты.
\- `/undo` - удалить последнюю трату.
\- `/budget <категория> <сумма>` - месячный лимит по категории.
\- `/budgets` - лимиты и остатки за текущий месяц.

При превышении лимита бот предупреждает в ответе на добавление траты.

## API

\- `POST /expenses` - `{user_id, amount, category}`, сохраняет расход (в ответе - статус лимита).
\- `GET /stats?user_id=&period=` - сумма по категориям за период.
\- `GET /expenses?user_id=&limit=` - последние траты.
\- `DELETE /expenses/last?user_id=` - удалить последнюю трату.
\- `POST /budgets` - `{user_id, category, limit}`, задать/обновить лимит.
\- `GET /budgets?user_id=` - лимиты, потрачено и остаток за месяц.
\- `GET /health` - проверка.
